Kirim SMS dari WordPress, kirim SMS pemberitahuan kepada user / member tentang posting terbaru, widget kirim SMS gratis bagi pengunjung web/blog.

== Description ==

Zenziva SMS memungkinkan anda sebagai administrator untuk dapat mengirim SMS kepada user/member web/blog anda. Kirim SMS ke satu, ke semua user
atau ke nomor tertentu. Anda juga dapat menyediakan widget Kirim SMS Gratis bagi para pengunjung web/blog anda, sehingga dapat meningkatkan trafik
kunjungan web/blog anda.

*   Kirim SMS ke semua user, ke grup tertentu, kesatu nomor.
*   Kirim SMS pemberitahuan mengenai posting terbaru blog anda.
*   Widget kirim SMS gratis untuk pengunjung web/blog anda.
*   Hanya untuk provider Indonesia ( Just for Indonesian provider )

== Installation ==

1. Install plugin zenzivasms.zip
1. Aktifkan plugin
1. Aktifkan widget ZenzivaSMS untuk mengaktifkan fitur Kirim SMS pada halaman.

== Frequently Asked Questions ==

= Bagaimana menggunakan plugin ini ? =

Anda terlebih dahulu harus mempunyai account di zenziva.com. Untuk daftar klik <a href="http://www.zenziva.com/harga/" target="_blank">disini</a>

= Dimana untuk mendapatkan http Api ? =

Setelah mempunyai account di zenziva.com, silahkan login dan masuk ke menu SMS SETTING > API SETTING atau lihat pada menu ZenzivaSMS pada WordPress.
